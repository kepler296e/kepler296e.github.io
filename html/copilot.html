GitHub Copilot

GitHub Copilot es una herramienta de inteligencia artificial basada en la nube desarrollada por GitHub y OpenAI para ayudar a los usuarios de Visual Studio Code, Visual Studio, Neovim y entornos de desarrollo integrado (IDE) de JetBrains mediante el autocompletado de código. Actualmente disponible por suscripción para desarrolladores individuales, la herramienta fue anunciada por primera vez por GitHub el 29 de junio de 2021 y funciona mejor para los usuarios que codifican en Python, JavaScript, TypeScript, Ruby y Go.
